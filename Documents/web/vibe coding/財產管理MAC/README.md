# 財產管理系統

一個使用 HTML + CSS + jQuery 開發的現代化財產管理系統，支援本地儲存和 Google Sheets 整合。

## 📋 功能特色

- ✅ 完整的資產管理（新增、編輯、刪除、查看）
- 🔍 即時搜尋功能
- 🎯 多條件篩選（財產單位、保管人、地點）
- 📊 統計儀表板（總資產數量、總金額、資產項目）
- 💾 本地儲存或 Google Sheets 同步
- 📥 匯出 Excel 功能
- 📱 響應式設計，支援手機和平板
- 🎨 現代化美觀的介面設計

## 🚀 快速開始

### 方法一：使用本地儲存（推薦新手）

1. 直接開啟 `index.html` 檔案
2. 資料會自動儲存在瀏覽器的本地儲存中
3. 已包含初始測試資料

### 方法二：整合 Google Sheets

1. **建立 Google Cloud 專案**
   - 前往 [Google Cloud Console](https://console.cloud.google.com/)
   - 建立新專案
   - 啟用 "Google Sheets API"

2. **建立 API 金鑰**
   - 在專案中建立憑證 → API 金鑰
   - 複製 API 金鑰

3. **準備 Google Sheets**
   - 建立新的 Google 試算表
   - 工作表名稱設為「財產清單」
   - 第一列輸入欄位標題：
     ```
     財產編號 | 財產名稱 | 規格/型號 | 數量 | 財產單位 | 購入/申請日期 | 取得日期 | 取得金額 | 保管單位 | 保管人 | 所在地點 | 供應商
     ```
   - 設定權限為「知道連結的任何人都可以檢視」
   - 複製試算表 ID（網址中的那串文字）

4. **設定配置**
   - 開啟 `config.js`
   - 填入你的 `SPREADSHEET_ID` 和 `API_KEY`
   - 將 `USE_LOCAL_STORAGE` 改為 `false`

## 🎯 使用說明

### 查看資產
- 主畫面會顯示所有資產列表
- 可以看到完整的資產資訊

### 搜尋資產
- 在搜尋框中輸入關鍵字
- 會即時搜尋所有欄位

### 篩選資產
- 使用下拉選單按財產單位、保管人或地點篩選
- 可以組合多個條件

### 新增資產
1. 點擊「新增資產」按鈕
2. 填寫資產資訊
3. 點擊「儲存」

### 編輯資產
1. 在資產列表中點擊「編輯」按鈕
2. 修改資訊
3. 點擊「儲存」

### 刪除資產
1. 在資產列表中點擊「刪除」按鈕
2. 確認刪除

### 匯出資料
- 點擊「匯出Excel」按鈕
- 會下載一個包含所有資產的 Excel 檔案

## 📌 注意事項

1. **本地儲存模式**：
   - 資料儲存在瀏覽器中
   - 清除瀏覽器資料會遺失所有記錄
   - 不同瀏覽器的資料不會同步

2. **Google Sheets 模式**：
   - 需要網路連線
   - 僅支援讀取（寫入需要 OAuth 認證）
   - 建議使用本地儲存模式進行編輯

3. **資料備份**：
   - 定期使用「匯出Excel」功能備份資料
   - 建議每週備份一次
